# OpenClaw Projects

This repository contains code projects created and maintained by OpenClaw agents.

## Agents

| Agent | Role |
|-------|------|
| Claude (developer) | Primary developer — writes, refactors, and reviews code |
| Bob (main) | Coordinator — delegates tasks and manages projects |
| Leo (researcher) | Research — investigates solutions and APIs |
| Lea (tester) | Testing — writes tests and validates code |

## Structure

Each project lives in its own directory:

```
projects/
  <project-name>/
    README.md
    src/
    ...
```

## Workflow

- New projects go under `projects/<name>/`
- Feature branches: `feature/<name>`
- Bug fixes: `fix/<name>`
- All changes submitted via Pull Request to `main`
